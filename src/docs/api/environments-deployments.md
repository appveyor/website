## Add environment

Request:

    POST /api/environments

Request body:

```text
{
   "name":"production",
   "provider":"FTP",
   "settings":{
      "providerSettings":[
         {
            "name":"server",
            "value":{
               "value":"ftp.myserver.com",
               "isEncrypted":false
            }
         },
         {
            "name":"username",
            "value":{
               "value":"ftp-user",
               "isEncrypted":false
            }
         },
         {
            "name":"password",
            "value":{
               "value":"password",
               "isEncrypted":true
            }
         }
      ],
      "environmentVariables":[
         {
            "name":"my-var",
            "value":{
               "value":"123",
               "isEncrypted":false
            }
         }
      ]
   }
}
```

    POST /api/deployments

```text

    "environmentName": "environment-to-deploy",
    "accountName": "your-account-name",
    "projectSlug": "project-slug-from-url",
    "buildVersion": "1.2.0",                      # build to deploy
    "buildJobId": "sfke9239ydzf",                 # optional job id with artifacts if build contains multiple jobs

       "server": "myserver.com",
       "another_var": "another value"

}
```

### Cancel deployment

Request:

    PUT /api/deployments/stop

Request body:

```text
{
    deploymentId: 123
}
```

Response: 204
