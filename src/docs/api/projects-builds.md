## Add project

Request:

    POST /api/projects

Request body:

```text
{
   "repositoryProvider":"gitHub",
   "repositoryName":"FeodorFitsner/demo-app"
}
```

Where `repositoryProvider` is one of:

* `gitHub`
* `bitBucket`
* `vso` (Visual Studio Online)
* `gitLab`
* `kiln`
* `stash`
* `git`
* `mercurial`
* `subversion`

Response:

```text
{
   "projectId":43682,
   "accountId":2,
```
